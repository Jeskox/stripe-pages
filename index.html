Stripe Payment Demo

These are the helper pages used in the checkout flow.

  • Success page — shown after a successful payment. Replace EXAMPLE with a real session ID in tests.
  • Cancel page — shown if the user cancels the payment.

Use the following in your Stripe session config:

success_url: https://jesko-lovat.vercel.app/success.html?session_id={CHECKOUT_SESSION_ID}
cancel_url: https://jesko-lovat.vercel.app/cancel.html
